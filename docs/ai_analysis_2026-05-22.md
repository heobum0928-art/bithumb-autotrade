## 2. 펌핑 패턴 인사이트

### 📈 **펌핑 이벤트 종합 분석**

```
총 펌핑 이벤트: 377회
평균 펌핑폭: 5.01%
평균 거래량 증가: 25.4배
```

### 🎯 **핵심 패턴**

| 지표 | 수치 | 의미 |
|------|------|------|
| **Pullback률** | 86% | 펌핑 후 86%가 2% 이상 되물린다 |
| **평균 5분 수익률** | **-2.39%** | 🔴 **펌핑 후 5분 평균이 음수** |
| **Bounce률** | 39% | 100회 중 39회만 반등 성공 |
| **실제 성과** | 24.6% 승률 | 펌핑 포착해도 이익화 실패 |

### ⚠️ **위험 신호**
```
펌핑 탐지 → 기대심리로 진입
→ 86% 확률로 2% 되물림 발생
→ -1.5% 손절 자동 발동
→ 결국 손절 손실

실제 데이터:
- SWAP: max_pnl +1.9% → 실현 -3.54% (손절)
- BOB: max_pnl +2.3-2.6% → 실현 -2.05~-2.15% (손절)
```

### 💡 **개선 방향**
```
문제: 펌핑 진입 직후 급락에 대응 불가

해결안 1) 동적 손절
- 펌핑 직후 1분: -0.8% (좁음)
- 펌핑 후 2-3분: -1.2% (중간)
- 펌핑 후 4분+: -1.5% (현재)

해결안 2) 펌핑 강도별 선별
- 거래량 20배 미만: 진입 금지
- 25-30배: -1.2% 손절
- 30배 이상: -1.5% 손절

해결안 3) 5분 누적 수익률 기준
- 펌핑 후 5분 내 -1% 이하면 자동 손절
```

## 3. 눌림목(Pullback) 전략 실현 가능성

### 📊 **데이터 기반 평가: ⚠️ 낮음 (현재 시스템으로는 부적합)**

### 문제점 분석

#### **① 진입 타이밍 문제**
```
필요한 조건:
1. 강한 상승 (펌핑 5% 이상) ✅ 평균 5.01% (충족)
2. 완전한 되물림 (2% 이상) ✅ 86% (충족)
3. 바닥에서 반등 신호 감지 ❌ 39% 성공률 (미달)
4. 반등 후 지속성 ❌ 평균 -2.39% (역방향)

→ 이상적 눌림목은 거래량 회복 + RSI 저점(30-40) + MACD 상향
→ 현재 봇은 이런 신호 없음
```

#### **② 실제 거래 결과**
```
최근 7일 분석:
- 펌핑 후 진입: 13건
- 이익 거래: 3건 (23%)
- 손실 거래: 10건 (77%)

손실 패턴:
1. SWAP: 진입 5분 후 급락 (-3.54%)
2. BOB: 진입 4분 후 급락 (-2.15%)
3. UP: 진입 8분 후 급락 (-2.32%)

→ 모두 되물림 이후 재하강 패턴
→ 눌림목 반등이 아닌 이중바닥(Double Bottom) 패턴
```
