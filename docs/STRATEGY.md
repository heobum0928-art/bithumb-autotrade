# STRATEGY.md — 전략 설계 및 파라미터 변경 이력

## 핵심 전략
60초 내 가격 +3% + 거래량 5배 폭발 감지 → 시장가 매수 → 트레일링 스탑

## 파라미터 결정 이유

| 파라미터 | 현재값 | 변경 이력 | 이유 |
|---------|-------|----------|------|
| VOLUME_MULT | 5x | 5x → 7x → 5x | 7x는 진입 기회 너무 희귀. RSI/MACD 복합 조건으로 보완 |
| INITIAL_STOP_PCT | -3% | -5% → -3% | MOVE 펌프덤프(-9,713원) 이후 강화 |
| 확인 딜레이 | 30초 | 12초 → 30초 | 펌프덤프는 20초 내 꺼짐 |
| TP_HALF | 2% | 6% → 2% | +6% 기다리다 되돌리는 케이스 많음 |
| HOLD_MIN_SEC | 600 | 신설 | 펌프덤프 방어, 수익구간은 조기트레일로 보완 |
| PRE_ENABLED | OFF | ON → OFF | 22건 14% 승률, -23,783원 누적 손실 |

## 필터 구조
1. BTC 하락 필터 (-1.5% 이상 시 진입 차단)
2. 거래량 지속성 필터 (이전 30초도 최소 30만원/분 이상 활성)
3. RSI 필터 (45~75 구간만 진입, 과열/침체 차단)
4. MACD 필터 (하락 추세 구간 차단, 데이터 부족 시 통과)
5. BB%B 필터 (> 1.0 차단 — 밴드 돌파 = 이미 과열, 되돌림 위험)
6. 거래량 상한 필터 (> 15배 차단 — 극단적 스파이크는 펌프덤프 패턴)
7. 호가 불균형 필터 (매수/매도 비율 1.5 이상)
8. 체결강도 필터 (매수틱 비율 60% 이상)
9. 30초 확인 딜레이 (가격 -1% 이상 하락 시 취소)
