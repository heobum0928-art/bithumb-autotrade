## 핵심 전략 (2026-05-25 저녁 전환)

### 전략 방향 전환 이유
펌핑 추격 전략의 구조적 한계 확인:
- 펌핑 감지 = 이미 가격이 올랐다는 뜻 → 감지 시점 = 고점 근처
- 필터 강화 → 좋은 코인도 차단 (WNCG 오늘 3번 차단)
- 결론: 파라미터 조정으로 해결 불가. 전략 구조 자체를 바꿔야 함

### 신규: 과매도 반등 전략 (메인)
```
[전체 마켓 폴링] → [24h +30% 코인 후보 풀 (5분 갱신)]
→ [3분봉 RSI 모니터링 (30초 주기)]
→ [RSI < 25 → 반등 대기]
→ [RSI ≥ 30 + MACD↑ + EMA9↑ → 매수]
→ [트레일링 스탑 3% (발동 +2.5%) + 익절 5%]
```

### 유지: IMMEDIATE_ENTRY_COINS (보조)
```
META, WNCG → 펌핑 감지 시 즉시 진입 (실거래 검증 완료)
```

### 비활성화 (2026-05-25)
```
PULLBACK_ENABLED = False  # 눌림목 추격 중단
→ 이유: C98 사례 — 눌림목 진입도 손실 지속, 과매도 반등으로 집중
```
