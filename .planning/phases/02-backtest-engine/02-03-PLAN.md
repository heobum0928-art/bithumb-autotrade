    2. `ev_ci(pnls: list[float]) -> tuple[float, float, float]`:
       ```python
       def ev_ci(pnls: list[float]) -> tuple[float, float, float]:
           """거래별 손익률 리스트 -> (EV, CI 하한, CI 상한). 정규근사 (D-12)."""
           n = len(pnls)
           ev = statistics.mean(pnls)
           if n < 2:
               return ev, ev, ev      # 표본 1건이면 CI 정의 불가
           se = statistics.stdev(pnls) / (n ** 0.5)   # 표본표준편차(n-1) 사용
           return ev, ev - Z_95 * se, ev + Z_95 * se
       ```
       주의: `stdev`(n-1 표본표준편차) 사용 — `pstdev` 금지.

    3. `max_drawdown(pnls_ordered: list[float]) -> float`:
       ```python
       def max_drawdown(pnls_ordered: list[float]) -> float:
           """시간순 정렬된 손익률 리스트 -> MDD (양수). 누적손익 단순합 모델."""
           equity = peak = mdd = 0.0
           for r in pnls_ordered:
               equity += r
               peak = max(peak, equity)
               mdd = max(mdd, peak - equity)
           return mdd
       ```

    4. `compute_metrics(trades: list[dict]) -> dict`:
       - 빈 리스트면 `{"count": 0}` 반환
       - `trades`를 `entry_ts` 기준 정렬해 `pnls_ordered` 추출 (MDD는 순서 의존)
       - `wins = [t for t in trades if t["pnl_pct"] > 0]`
       - 반환 dict: `count`, `win_count`, `win_rate` (= win_count/count), `ev`, `ci_low`, `ci_high`, `mdd`, `sample_warning` (= count < 2 또는 count < 30 시 True)
       - `ev`/`ci_low`/`ci_high`는 `ev_ci([t["pnl_pct"] for t in trades])` 결과
